# Project Progress Tracker

A web-based application built with Angular for users and admins to collaboratively report and monitor the progress of various projects in real-time.

## Features

-   Role-based access control (Super Admin, Admin Kantor, Admin Lapangan).
-   Real-time project progress tracking with percentage bars.
-   Detailed project view with historical updates.
-   Daily report submission with photo evidence (camera access or file upload).
-   User management for creating new users.
-   Project creation and management.

## How to Run Locally

Follow these steps to get the development environment running.

### Prerequisites

-   [Node.js and npm](https://nodejs.org/)
-   [Git](https://git-scm.com/)

### Installation & Setup

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git
    ```
    (Replace `YOUR_USERNAME` and `YOUR_REPOSITORY_NAME` with your actual GitHub details)

2.  **Navigate to the project directory:**
    ```sh
    cd project-progress-tracker
    ```

3.  **Install dependencies:**
    This command will download all the necessary packages for Angular.
    ```sh
    npm install
    ```

4.  **Run the development server:**
    ```sh
    npm start
    ```
    The application will be available at `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
